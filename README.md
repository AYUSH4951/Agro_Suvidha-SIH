Agro Suvidha is a user-friendly web application developed for the Smart India Hackathon (SIH).  
It provides essential agricultural information such as soil health insights, mandi price updates, and farming solution guides.  
The project is built using HTML, CSS, and JavaScript and includes PWA features like offline access through a service worker.  
Pages such as Soil Health, Mandi Price, Solutions, and About Us allow easy navigation for farmers.  
The application also supports multilingual content using a JSON-based translation system.  
It is responsive and works smoothly on both mobile and desktop devices.  
This repository includes all UI files, assets, icons, and scripts required to run the project.  
To use the app, simply clone the repository and open `index.html` in any modern browser.  
Agro Suvidha aims to make agricultural information accessible, reliable, and simple for farmers to use.
